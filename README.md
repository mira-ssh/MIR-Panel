# 🚀 MIR Panel - Advanced DPI Evasion System

یک پنل تحت وب قدرتمند و سبک برای دور زدن پیشرفته‌ترین سیستم‌های فیلترینگ (DPI). این پروژه با استفاده از هسته Xray و تکنیک Fragmentation، ترافیک شما را تکه‌تکه کرده و مانع از شناسایی آن توسط فایروال‌ها می‌شود.

## ⚡ امکانات کلیدی (Features)
* **پنل مدیریت تحت وب:** رابط کاربری تمیز و ساده بدون نیاز به درگیری با ترمینال.
* **دور زدن DPI (Fragmentation):** تکه‌تکه کردن پکت‌های TLS برای مخفی ماندن از دید سیستم‌های فیلترینگ.
* **پروتکل VLESS + WebSocket:** امکان اتصال به کلادفلر و استفاده از IP تمیز (CDN).
* **تولید خودکار کانفیگ:** ساخت سریع لینک اتصال تنها با یک کلیک.
* **مانیتورینگ منابع:** نمایش زنده وضعیت CPU، RAM و سرویس‌ها.
* **سیستم عیب‌یاب خودکار (Watchdog):** بررسی مداوم و ری‌استارت خودکار سرویس در صورت قطعی.

برای نصب پنل روی سرور خام لینوکس (پیشنهاد می‌شود از Ubuntu 20.04 یا 22.04 استفاده کنید)، با کاربر `root` وارد سرور شده و دستور تک‌خطی زیر را در ترمینال کپی و اجرا کنید:
