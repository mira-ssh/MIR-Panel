# 🚀 MIR Panel - Advanced DPI Evasion System
A lightweight and powerful web panel for bypassing advanced DPI filtering systems. This project utilizes the Xray core and fragmentation techniques to obfuscate traffic.

## ⚡ Features
* **Web Management Panel:** Simple UI with no terminal dependency.
* **DPI Evasion (Fragmentation):** TLS packet fragmentation to hide from filtering.
* **VLESS + WebSocket:** Supports Cloudflare and Clean IPs.
* **Auto-Config Generator:** Create connection links with one click.
* **Watchdog System:** Automated service monitoring and restart.
